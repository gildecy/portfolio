Projeto 1

Descrição.

O projeto em HTML e CSS que calcula o IMC (Índice de Massa Corporal) é uma ferramenta interativa que permite aos usuários inserir seu peso e altura para calcular automaticamente seu IMC. O IMC é uma medida comumente usada para avaliar se uma pessoa está com peso adequado para sua altura. Com base nos valores fornecidos, o projeto exibirá o resultado do cálculo do IMC e fornecerá uma indicação se a pessoa está abaixo do peso, dentro do peso normal, com sobrepeso ou obesa. A interface do projeto é amigável e fácil de usar, incentivando os usuários a monitorar sua saúde e tomar decisões informadas sobre seu peso.
